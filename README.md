# joelnjoku.github.io

Research-grade portfolio starter with React + Vite + React Router.

## Structure

- `index.html` — Vite entry HTML
- `styles/` — global and route-level CSS files
  - `styles.css`
  - `about.css`
  - `contact.css`
  - `projects.css`
- `src/pages/` — route components
- `src/routes/router.jsx` — centralized route map
- `.github/workflows/ci.yml` — CI checks on push/PR

All pages and styles are intentionally empty to provide a clean starting point.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`
- `npm run preview`
